# Portfolio Project Guidelines

When updating the portfolio or adding new projects, follow these strict rules:

## 1. Project Screenshots
- **NEVER generate custom mockups** using AI tools (e.g., Image Generation).
- **ALWAYS use real screenshots** of the active website.
- Use the browser tool to navigate to the live URL and take a high-quality viewport screenshot.
- Save images to `src/assets/projects/` using descriptive names (e.g., `projectname-shot.png` or `.webp`).

## 2. Bento Card Structure
- Maintain the bento grid pattern.
- Ensure both Slovak (`index.html`) and English (`en/index.html`) versions are updated.
- Update the project counter (`#portfolio-counter`) and any relevant statistics in the "About" section.

## 3. SEO & Accessibility
- Ensure every new project card has proper `title` attributes on links and `alt` tags on images.
- Use semantic HTML.
