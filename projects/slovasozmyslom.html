Case study - WordPress / Gutenberg

Slová so zmyslom

Web pre slovenskú copywriterku, redaktorku, prekladateľku a lektorku angličtiny Zuzanu Kolačanovú. Cieľom bolo vytvoriť pokojný editorial web, ktorý pôsobí osobne, dôveryhodne a jasne vysvetľuje služby.

Typ projektuFiremný web / osobná značka
StackWordPress, Gutenberg, Blocksy, SEO
Otvoriť live web

Problém klienta

Klientka potrebovala nový web, ktorý nebude pôsobiť korporátne ani genericky. Bolo potrebné prehľadne rozdeliť služby, referencie, publikované texty, blog, kontakt a právne informácie tak, aby sa návštevník rýchlo zorientoval.

Čo bolo implementované

  • Kompletná WordPress štruktúra pripravená cez Gutenberg-kompatibilné page snippets.
  • Homepage, služby a cenník, referencie, blog a tipy na knihy, portfólio článkov, kontakt a ochrana osobných údajov.
  • Migrácia a reorganizácia pôvodného obsahu aj klientských dokumentov do jasnejších sekcií.
  • Custom typewriter-style logo a export transparentných PNG/SVG assetov.
  • Globálne CSS pre typografiu, spacing, karty, tlačidlá, responzívny layout a integráciu s Blocksy témou.
  • All in One SEO export konfigurácia: meta dáta, schema basics, social previews, sitemap a noindex pravidlá pre médiá a archívy.

Dizajnový smer

Finálny vizuál používa editorial feeling „writer’s desk“: teplé papierové tóny, jemnú grid textúru, tmavý atramentový text, tlmené zelené akcenty, serifové nadpisy a minimalistické UI. Výsledok sedí copywriter/editor značke bez toho, aby pôsobil stroho alebo šablónovo.

Výsledok

Web má jasnú prezentáciu osobnosti aj služieb, funkčné sekcie pre referencie a portfólio, pripravený blog/archive systém a SEO nastavenia pre ďalší rast obsahu.
