Case study - WooCommerce / WordPress

Modafinil CZ

Kompletný WooCommerce e-shop pre český trh, postavený podľa klientom zvoleného referenčného štýlu a upravený do vlastnej zelenej vizuálnej identity s českým obsahom.

Typ projektuWooCommerce e-shop
StackWordPress, WooCommerce, Blocksy, Gutenberg
Otvoriť live web

Problém klienta

Klient potreboval rýchlo spustiteľný český e-shop s produktmi, variantmi balení, zrozumiteľnou štruktúrou stránok a vizuálom, ktorý pôsobí výrazne, ale stále zostáva použiteľný pre nákupný proces.

Čo bolo implementované

  • WooCommerce produktové dáta pre 7 hlavných produktov vrátane variantov balení 10, 20, 30, 50 a 100 tabliet.
  • Zľavová logika pre väčšie balenia, krátke aj plné produktové popisy, detailné tabuľky a informácie o množstvách.
  • Gutenberg homepage so sekciami hero banner, benefit row, produktový grid, blog a pripravená footer štruktúra.
  • Custom CSS pre Blocksy/WooCommerce: produktové karty, archív, detail produktu, tlačidlá, typografia, spacing a mobilné zobrazenie.
  • Stránky pre kontakt, dopravu a platbu, košík, checkout, účet, o nás, právne stránky, cookies/privacy a úvod obchodu.
  • Custom PHP footer snippet s logom, kontaktom, top produktmi, užitočnými odkazmi, copyrightom a TH Studio kreditom.
  • SEO exporty pre All in One SEO, sitemap nastavenia, produktové SEO defaults a importné workflowy pre WooCommerce aj blog.

Dizajnový smer

Vizuál stavia na kontraste čistej e-shopovej štruktúry a výrazných zelených akcentov. Homepage používa veľký banner, jasné benefity, produktovú ponuku a praktické navigačné prvky, aby návštevník rýchlo pochopil sortiment aj možnosti nákupu.

Výsledok

Projekt dostal kompletný základ pre český WooCommerce predaj: pripravené produkty, stránky, importy, SEO nastavenia, platobné a dopravné informácie, vlastný footer a responzívny frontend.
