Projekt - WordPress / Kadence

Profi Výživa

WordPress web pre výživové poradenstvo, jedálničky na mieru a odborné konzultácie. Cieľom bolo vytvoriť pokojný, dôveryhodný web, ktorý jasne vysvetlí služby a návštevníka prirodzene dovedie k nezáväznému záujmu.

Zadanie klienta

Web mal prehľadne predstaviť výživové programy, online konzultácie, vstupnú diagnostiku a odborné materiály. Dôležité bolo, aby stránka nepôsobila ako tvrdý fitness predaj, ale skôr ako pokojná a praktická služba pre ľudí, ktorí chcú zmenu bez extrémov.

Rozsah práce

  • Príprava homepage vo WordPress/Gutenberg štruktúre s Kadence blokmi.
  • Vizuálne smerovanie: jemná zelená paleta, čisté karty, veľké nadpisy a dôraz na čitateľnosť.
  • Sekcie pre služby, spoluprácu, ukážky materiálov, certifikáty, FAQ a kontaktný formulár.
  • Samostatná stránka vstupnej diagnostiky s uprataným layoutom a návratom na hlavný web.
  • Právne a informačné stránky: obchodné podmienky, ochrana osobných údajov a cookies.
  • Globálne CSS pre Kadence/WordPress, responzívne zobrazenie, karty, tlačidlá a formuláre.
  • Code snippets pre akciové ceny, certifikátový lightbox, úpravu diagnostickej stránky a slovenské štítky Complianz.

Technické doladenie

Súčasťou práce bolo aj upratanie detailov okolo pluginov, zobrazenia ikoniek, cookies textov a finálnej kontroly pred odovzdaním. Projekt bol pripravený tak, aby sa dali neskôr dopĺňať ďalšie certifikáty, materiály alebo obsah bez prerábania celého dizajnu.

Výsledok

Profi Výživa získala prehľadný WordPress web so silnou úvodnou stránkou, jasným predstavením služieb, vstupnou diagnostikou, právnymi stránkami a technickým základom pripraveným na ďalšie rozširovanie.
