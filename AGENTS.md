# Agent Instructions

Speak English in chat.

When writing or editing public blog posts for this portfolio:

- Read `BLOG_WRITING_STYLE.md` first.
- Match Tomáš's cleaned-up voice: direct, practical, slightly informal, and based on real problems from work.
- Write public blog posts in Slovak unless the user asks otherwise.
- Do not make the text sound like a corporate agency blog or generic SEO filler.
- Do not copy raw typos into published text unless the user explicitly asks for that style.
- Do not publish private client details, admin screens, analytics screenshots, contact data, payment details, disputes, internal notes, or staging/GitHub workflow details.

When updating portfolio projects:

- Read `PORTFOLIO_GUIDELINES.md`.
- Use real screenshots, public-safe details, and avoid implying client endorsement unless explicitly approved.
