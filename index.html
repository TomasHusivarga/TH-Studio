FAQ

Časté otázky pred
začiatkom spolupráce.

Koľko stojí tvorba webstránky?

Jednoduchý 2-3 stránkový WordPress web začína od 400 €. Väčší firemný web alebo e-shop závisí od počtu stránok, produktov, funkcií, obsahu a napojení.

Ako dlho trvá vytvorenie webu?

Menší 2-3 stránkový web viem pripraviť približne do 2 dní. Väčší firemný web trvá zvyčajne okolo 4 dní a WooCommerce e-shop približne týždeň.

Robíš aj WooCommerce e-shopy?

Áno. Nastavujem produkty, kategórie, platobné metódy, emaily, dopravu, košík a pokladňu podľa toho, čo e-shop reálne potrebuje.

Pomôžeš s obsahom a SEO?

Áno. Viem pomôcť so štruktúrou obsahu, textami, nadpismi, meta popismi, technickým SEO, rýchlosťou a základným nastavením pre Google.

Vieš upraviť existujúci WordPress web?

Áno. Môžem opraviť chyby, pridať sekcie, upraviť Elementor alebo Divi, zrýchliť načítanie a vyriešiť problémy po aktualizáciách pluginov.

Najrýchlejší odhad

Pošli mi typ webu, počet stránok a či už máš texty. Väčšinou viem rýchlo povedať, či stačí menší web, úprava alebo e-shop.
