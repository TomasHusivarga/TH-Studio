# TH Studio SEO Keyword Map

Use this map when editing service pages, homepage sections, blog cards, titles, meta descriptions and internal links. The goal is natural Slovak wording, not keyword stuffing.

## Homepage

- Primary intent: tvorba webstránok a e-shopov pre firmy
- Main phrases: tvorba webstránok, WordPress webstránky, WooCommerce e-shop, weby pre firmy, Slovensko
- Page role: overview and trust page that sends people to service pages, portfolio and contact

## Tvorba webstránok

- Primary intent: someone wants a new website for a business or service
- Main phrases: tvorba webstránok pre firmy, webstránka pre živnostníka, cena webstránky, tvorba WordPress webstránok
- Useful supporting topics: 2-3 page website, content structure, mobile layout, basic SEO, contact form
- Main URL: `/TH-Studio/sluzby/tvorba-webstranok.html`

## Tvorba WooCommerce e-shopu

- Primary intent: someone wants an e-shop; WooCommerce is the implementation keyword, not the main buyer wording
- Main phrases: tvorba e-shopu, tvorba e-shopu cena, tvorba WooCommerce e-shopu, WordPress e-shop, platby a doprava WooCommerce
- Useful supporting topics: products, variants, cart, checkout, shipping, payments, emails, product SEO
- Main URL: `/TH-Studio/sluzby/tvorba-eshopov.html`

## Úprava WordPress webu

- Primary intent: someone has an existing website or store that needs help
- Main phrases: oprava webstránky, úprava webu, oprava WordPress webu, WooCommerce problém, chyba po aktualizácii
- Useful supporting topics: plugin conflict, Elementor, Divi, Gutenberg, checkout, speed, mobile layout, temporary access
- Main URL: `/TH-Studio/sluzby/uprava-wordpress-webu.html`

## Blog Support

- Pricing article intent: koľko stojí webstránka, cena webstránky, WordPress web cena, tvorba web stránky cena
- Repair article intent: WooCommerce problém oprava, oprava e-shopu, nefunguje WooCommerce
- Trust article intent: prečo firma potrebuje web, webstránka pre podnikanie
