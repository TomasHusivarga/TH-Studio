How much does a website cost?

Short answer: a smaller 2-3 page website starts from 400 EUR. Longer answer: it depends on what the website needs to do, how many pages it has and how much material is already prepared. I usually build these websites in WordPress.

What a website from 400 EUR means

This price makes sense for a smaller business or personal website with a clear scope. Usually that means a homepage, a service or basic subpage, contact, responsive layout, form and basic SEO setup.

It is not only “installing a template”. The website still needs to explain the offer, make the next step clear and avoid hiding basic contact or service information.

What changes the price most

The price grows when there are more pages, more copywriting, special sections, language versions, blog setup, booking systems, integrations or more revision rounds. Sometimes the content work is bigger than the WordPress setup itself.

If you already have a logo, photos, copy and a clear offer, the project moves faster. If the offer has to be reorganized first, that is a larger scope.

How much does an e-shop cost?

A WooCommerce store can start near a smaller website, but it quickly depends on products. Product variants, shipping, payments, emails, legal/info pages, imports and order testing all affect the real scope.

For an e-shop, I first look at what will be sold, how many products there are and how simple or complex the checkout should be.

Domain, hosting and email are separate

Website pricing should also include running costs. Domain, hosting, email, premium plugins or paid services are usually separate costs. They are not payment for my work, but the website may need them to run properly.

For a smaller WordPress website, this can usually stay reasonable without unnecessary subscriptions. The important part is knowing what is one-time work and what is recurring cost.

Why the cheapest website is not always cheaper

The cheapest offer can look good until you find out that mobile layout, basic SEO, content structure, speed, forms, security or editing access were not really handled. Then the cheap website often needs to be fixed again.

That is why I prefer a smaller but usable scope. A website does not need everything. It needs a solid base that can be expanded later.

How to get a realistic estimate

Send the website type, number of pages, whether you already have copy and whether this is a new website or an edit of an existing WordPress site. From that I can usually tell whether it fits a smaller website from 400 EUR, hourly edits or a custom quote.
